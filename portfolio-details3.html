Détails du travail

La base de ce projet était de refactorer le projet C que l'on avait fait en 2ième avec le langage C# pour un test d'entré pour un stage. Cela m'a permis d'apprendre le langage C# et le .NET par la même occasion.
Malgré le fait que je n'ai pas pu réaliser l'entierté du projet qui demandait faire fonctionner ce projet en ASP.net, j'ai réussi à réfactorer les fonctions principales du projet de 2ième tels que la simulation d'une course avec plusieurs voitures en multi-thread, générer des nombres aléatoires pour chaques tour, etc ...
